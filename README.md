## RecSys-youtube

🎬 YouTube 연관규칙 추천시스템
Global YouTube Statistics.csv 데이터를 기반으로 Apriori와 FP-Growth 두 가지 알고리즘으로 연관규칙(Association Rules)을 생성·비교하는 프로젝트입니다.
교육·실습 목적의 예제이지만 실제 데이터 파이프라인과 팀 협업 흐름을 고려해 구조화했습니다. <br> <br> <br>


⚙️ 요구 사항
패키지 | 버전 이상
Python | 3.9
pandas | 2.0
mlxtend | 0.23
